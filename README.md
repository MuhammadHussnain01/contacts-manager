# 📇 Contact Manager

A simple command-line Contact Manager built with **Python** that lets you add, search, and delete contacts. All contact information is stored in a CSV file, making it a great beginner project for learning file handling and data management.

## ✨ Features

* ➕ Add new contacts
* 🔍 Search contacts by phone number
* 🗑️ Delete contacts
* 💾 Store contacts in a CSV file

## 🛠️ Built With

* Python 3
* Built-in `csv` module

## 🚀 Getting Started

```bash
git clone https://github.com/MuhammadHussnain01/contacts-manager.git
cd contacts-manager
python main.py
```
